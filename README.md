# Dog vs Cat Image Classification

This project classifies images of cats and dogs using a deep learning model.

## Dataset
The dataset is downloaded from Kaggle using kagglehub.

## How to Run
1. Open the notebook in Google Colab or Kaggle
2. Run all cells from top to bottom
